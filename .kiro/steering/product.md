# Product Overview

A dynamic professional portfolio website that reads all content from a PDF file (Profile.pdf). The site features a modern, responsive design and includes an AI-powered chat assistant that can answer questions about the portfolio owner's career and experience.

## Key Features

- PDF-driven content: All profile data (name, experience, education, skills) is extracted from Profile.pdf
- AI chat assistant: OpenRouter-powered chatbot that acts as a digital twin
- Responsive design: Modern UI that works across devices
- Multiple sections: Home, About, Career timeline, Portfolio, Contact
- Zero hardcoded content: Update the PDF, restart the app - no code changes needed

## Target Deployment

- Primary: Render.com (free tier)
- Local development on port 8080
- Environment-based configuration for API keys
